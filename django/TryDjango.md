
# MVC & MTV
 1. Model
   * 안전하게 데이터를 저장
   * integrity, consistency, queries & mutations
 2. View
   * 데이터를 적절하게 유저에게 
   * presentation, assets & code
 3. Controller
   * 사용자의 입력과 이벤트에 반응하여 model과 view를 업데이트
   * receive, interpret & validate input, create & update views, query & modift models
